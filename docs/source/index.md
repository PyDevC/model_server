Model Server documentation
===================================

**Model Server**: is a LLM inferencing tool, build using huggingface and PyTorch,
It fulfiles the general purpose of running models as a alternative to machines 
that can't install ollama or llama.cpp.

We tried to make it completely cross-platform and available if the set of tools and builds are right

**Small developement needed**: Need to build a small cli binary to run all
